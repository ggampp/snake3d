# Créditos de recursos

## Texturas dos planetas

As superfícies dos planetas do jogo (Terra, Marte, Lua, Vênus, Júpiter e
Mercúrio) usam mapas equiretangulares do **Solar System Scope**.

- Fonte: https://www.solarsystemscope.com/textures/
- Obtidas via Wikimedia Commons: https://commons.wikimedia.org/wiki/Category:Solar_System_Scope
- Licença: **Creative Commons Attribution 4.0 International (CC BY 4.0)** —
  https://creativecommons.org/licenses/by/4.0/

> As imagens são baseadas em dados de imagens da NASA. É permitido usar, adaptar
> e compartilhar as imagens, inclusive para fins comerciais, desde que a fonte
> (Solar System Scope) seja creditada.

Arquivos em `public/textures/planets/`:

| Arquivo        | Corpo     |
| -------------- | --------- |
| `earth.jpg`    | Terra     |
| `mars.jpg`     | Marte     |
| `moon.jpg`     | Lua       |
| `venus.jpg`    | Vênus     |
| `jupiter.jpg`  | Júpiter   |
| `mercury.jpg`  | Mercúrio  |

Todas em resolução 2k (2048×1024).
